## 1. 改造概述

### 1.1 原系统架构

| 组件 | 原系统 |
|------|--------|
| 后端框架 | ruoyi-vue-pro (Spring Boot 3 + MyBatis Plus) |
| 管理后台前端 | yudao-ui-admin-vue3 (Vue 3 + Element Plus) |
| 用户端前端 | yudao-mall-uniapp (uni-app，多端) |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis 7 |

### 1.2 改造目标

在原有商城系统基础上，实现社区团购系统，支持：
- 团购活动发布与管理
- 用户参团/开团
- 自提点管理
- 多租户数据隔离（每个商家独立租户）
- 管理员/商家角色分离

### 1.3 改造范围

| 模块 | 改造内容 |
|------|----------|
| 数据库 | 新增 4 张团购业务表，18 张表添加 tenant_id，8 张缺失表 |
| 后端 | 新增 group-buy Activity/Record/Member 模块，多租户改造 |
| 前端（管理端） | 首页实时数据、菜单精简、角色权限、快捷入口修正 |
| 前端（用户端） | 替换为 yudao-mall-uniapp（官方 uni-app 项目） |

### 1.4 项目 Git 结构

| 路径 | 类型 | 来源 |
|------|------|------|
| `backend/ruoyi-vue-pro` | submodule | github.com/YunaiV/ruoyi-vue-pro |
| `frontend/yudao-ui-admin-vue3` | submodule | github.com/yudaocode/yudao-ui-admin-vue3 |
| `frontend/yudao-mall-uniapp` | submodule | github.com/yudaocode/yudao-mall-uniapp |

## 5. 多租户架构

### 5.1 设计模式：模型 A（完全隔离）

每个商家 = 独立租户。会员/商品/订单/拼团数据完全隔离。

### 5.2 隔离机制

- `TenantLineInnerInterceptor`：对所有 `TenantBaseDO` 实体自动注入 `WHERE tenant_id = ?`
- `TenantSecurityWebFilter`：阻止非管理员直接改 `tenant-id` 头跨租户访问

### 5.3 已创建租户

| 租户 ID | 名称 | 管理员账号 | 角色 |
|---------|------|------------|------|
| 1 | 默认租户 | admin | super_admin |
| 162 | 社区团购A | zhangsan01 | tenant_admin + merchant |
| 163 | 社区团购B | lisi02 | tenant_admin + merchant |

### 5.4 管理员跨租户切换

- 管理员可通过 `tenant-id` 请求头切换访问目标租户数据
- 商家不可跨租户访问（返回 403）
- 前端 `TenantVisit` 组件需 `VITE_APP_TENANT_ENABLE=true` + `system:tenant:visit` 权限

### 5.5 角色分离

| 角色 | code | 菜单数 | 模块 |
|------|------|--------|------|
| 超级管理员 | super_admin | 全部 | 5 个模块 |
| 商家 | merchant | 38 项 | 3 个模块（商城+会员+支付） |

## 6. 遇到的问题与解决方案

### 6.1 构建与环境

| # | 问题 | 解决方案 |
|----|------|----------|
| 1 | FUSE overlay 导致 `mvn clean` 失败 | 改用 `mvn install` 跳过 clean |
| 2 | 前端容器 CORS（host.containers.internal 不可解析） | 使用 `.env.local` 配置 `localhost:48080` |
| 3 | MySQL 字符集 latin1 导致菜单乱码 | 添加 `custom.cnf` 强制 utf8mb4 |

### 6.2 多租户

| # | 问题 | 解决方案 |
|----|------|----------|
| 4 | 18 张表缺少 `tenant_id` | 批量 ALTER TABLE 添加 |
| 5 | 缺失 8 张表 | 创建建表语句 |
| 6 | GroupBuy DO 继承 BaseDO 无法隔离 | 改为继承 TenantBaseDO |
| 7 | 商家角色跨租户访问 | 添加 TenantSecurityWebFilter |

### 6.3 代码 Bug

| # | 问题 | 解决方案 |
|----|------|----------|
| 8 | 统计页 NPE（无 times 参数） | 3 个 Controller 添加 null 检查 + 默认值 |
| 9 | 首页数字写死 | 改为 4 个实时 API 调用 |
| 10 | 快捷入口 3 个 404 | 修正为叶子路由路径 |
| 11 | member/summary 字段名不对 | `registerMemberCount` → `userCount` |

### 6.4 权限与菜单

| # | 问题 | 解决方案 |
|----|------|----------|
| 12 | 前端菜单白名单缺少前缀 | 添加 `/system`、`/infra` |
| 13 | group-buy 权限未注册 | 创建 `promotion:group-buy:query` 等 4 个菜单权限 |
| 14 | 租户套餐含无用菜单 | 清理租户套餐 111 的菜单关联 |
